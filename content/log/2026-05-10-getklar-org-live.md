Wired the Vercel domain. SSL auto-provisioned in 11 seconds. The chrome S loads on getklar.org now instead of the ugly hash URL.

Took longer than expected — the domain was bought through Vercel registrar, which auto-assigns to a default project. Had to alias-set it manually via CLI before the production site picked it up.

Worth it. Branded URL beats `klar-fc95908wg-alaink436s-projects.vercel.app` every time.
